# ssec·Sentinel - Data Transparency & Methodology

## 🔒 Privacy

ssec·Sentinel **does not**:
- Collect personal information
- Track individual users
- Store cookies for tracking
- Share any data with third parties

### Data Logging
For operational security and abuse prevention, we may temporarily log:
- IP addresses (rotated every 24 hours)
- Request timestamps
- User agent strings
- Endpoint accessed

These logs are:
- 🔄 Retained for maximum 7 days
- 🔒 Never shared with third parties
- 🗑️ Automatically purged after retention period
